js中substr与substring区别

发表于 2020-03-13 | 更新于: 2020-03-13 | 分类于 笔记 ， js

概述

主要作用：截取并返回新字符串

  • substr
     语法：'被截取的字符串'.substr(start[,length])
     说明：从 start 下标索引开始往后截取指定长度的字符。
  • substring
     语法：'被截取的字符串'.substring(start[,end])
     说明：从 start 下标索引开始截取到 end 下标索引。

注意事项

  1. 当只有一个参数时，两个函数功能是一样的，都返回 start 到末尾的字符串。
  2. 如果 substr 的 length 参数为 0 或负数，返回空字符串。而 substring 的 end 为 0 或负数，返回 start 下标的字符串
